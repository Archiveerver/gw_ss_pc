Coke ScreensaverInstallation instructions1.  Save the Stuffit archive file to your hard drive. To uncompress, double-click on the file. If Stuffit Expander does not launch automatically, you may need to install this software at http://www.aladdinsys.com/expander/expander_mac.html. 2.  Drop the Screensaver file on top of your System Folder.  A prompt will tell you that the file you just droped is a control panel and ask you if you'd like the computer to place it in the Control Panels Folder.  Hit "OK"3. The next time you restart your computer, your Coca-Cola screensaver should be in place!
